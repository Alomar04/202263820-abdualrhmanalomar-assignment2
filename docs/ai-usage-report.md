# AI Usage Report

## Tools Used & Use Cases
The main AI tool used in this assignment was ChatGPT. It was used within acceptable coursework uses, including code generation or suggestions, debugging help, code review or improvements, writing documentation, and UI/UX or accessibility suggestions.

In this project, AI was used to:
- generate an initial structure for the HTML, CSS, and JavaScript files
- suggest how to implement tabbed navigation without reloading the page
- help with the dark/light theme toggle and `localStorage` persistence
- support form validation logic and user feedback messages
- suggest improvements for layout, responsiveness, and interaction design
- help draft the README, technical documentation, and this AI usage report

## Benefits & Challenges
One benefit of using AI was speed. It helped produce a working first draft of the portfolio much faster, which gave more time for refinement and testing. It also helped turn the assignment requirements into concrete frontend features more efficiently.

Another benefit was support during debugging and revision. AI was useful for suggesting possible fixes and improvements when adjusting the tab behavior, form validation, and theme logic.

The main challenge was that AI output was not always fully correct or fully appropriate. Some text sounded too generic, some project content did not match the real portfolio, and some behaviors still needed manual correction. Because of that, the AI output could not be accepted as-is and needed careful review and modification.

## Learning Outcomes
Using AI in this assignment supported learning rather than replacing it. It helped reinforce:
- how JavaScript can dynamically switch visible sections on a page
- how `localStorage` can be used to save theme preferences
- how client-side form validation and inline feedback work
- how CSS transitions and animations improve user experience
- how a small frontend project can be organized into separate files for structure, styling, behavior, and documentation

It also improved my ability to read, evaluate, and understand generated code. I had to make sure the code actually matched the assignment requirements before keeping it.

## Responsible Use & Modifications
I understood the AI-generated code before using it, and I edited or improved the output instead of submitting it unchanged. AI supported the development process, but it did not replace my own understanding or decision-making.

Example of changes I made after reviewing AI output was fixing the project links so they correctly open the Contact tab.

AI was used responsibly. It helped with code generation, debugging, review, documentation, and design suggestions, but the final submission was reviewed, corrected, and improved by me to ensure it matched the assignment and demonstrated real understanding.
