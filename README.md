## Project Description
This project upgrades a static portfolio into a modern interactive web application using HTML, CSS, and JavaScript. The website includes tabbed navigation for switching between portfolio sections without reloading the page, a dark/light theme toggle that persists with `localStorage`, a validated contact form with inline feedback, and smooth visual transitions that improve the user experience.

The implementation is designed to align directly with the assignment rubric by emphasizing clean structure, responsive layout behavior, readable code, accessibility-minded markup, and clear technical documentation.

## Features
- JavaScript tabbed navigation for `About`, `Projects`, and `Contact`
- Dark/light theme toggle with saved preference in `localStorage`
- Contact form validation for empty fields and invalid email format
- Inline error messages and a success confirmation message
- Fade-in tab transitions and hover animations on buttons and project cards
- Responsive layout for desktop, tablet, and mobile screens

## Setup Instructions
1. Download or clone the project folder.
2. Open the project directory.
3. Launch `index.html` in any modern web browser.

No build tools, package managers, or external setup steps are required because this is a static frontend project.

## How to Use
1. Click the navigation tabs to switch between sections.
2. Use the theme toggle button in the header to switch between light and dark mode.
3. Refresh the page to confirm that the selected theme is saved.
4. Open the `Contact` tab and test the form validation using empty fields, invalid email input, and valid input.

## Browser Compatibility
The project uses standard HTML5, CSS3, and vanilla JavaScript features that work across current versions of Chrome, Edge, Firefox, and Safari.
