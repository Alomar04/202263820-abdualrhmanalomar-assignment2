## Project Overview
This portfolio showcases Abdulrahman Alomar’s academic and personal work as a Junior SWE Student at KFUPM. The site highlights two AI-driven projects, presents a clean personal introduction, and provides a contact form. It is designed as a fast, responsive, and accessible single-page experience built with modern frontend tools.

## Features
- Dark Mode toggle with saved preference
- Time-based greeting (Morning/Afternoon/Evening)
- Responsive project grid for mobile and desktop

## Setup Instructions
Because this is a static website, setup is lightweight:

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```
2. Open the project folder.
3. Launch the site by opening `index.html` in your browser.

## Usage Guide
- **Theme Toggle:** Use the button in the navbar to switch between light and dark mode. Your preference is saved locally in the browser.
- **Contact Form:** Fill in the name, email, and message fields, then click “Send Message.” A confirmation alert appears to indicate submission.
