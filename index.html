Good day.

Upgrading a personal portfolio into an interactive web application.

I am a Software Engineering student at KFUPM, and this project was built for SWE-363 as part of moving from a static portfolio to a more interactive frontend experience. The goal was to keep the website personal and practical while adding features that improve usability.

This version uses JavaScript to switch between sections without reloading the page, remember the selected theme with localStorage, and validate the contact form with clear feedback. It reflects both the course requirements and the type of project work I am interested in building.
